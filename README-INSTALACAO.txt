# Nova Sabará Mensalidades — Pacote PWA

Suba todos estes arquivos na raiz do repositório/pasta `NovaSabara` no GitHub Pages.

Arquivos essenciais:
- `index.html`
- `manifest.webmanifest`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`
- `apple-touch-icon.png`
- demais ícones gerados

Depois de subir:
1. Abra `https://ceducunha-alt0609.github.io/NovaSabara/`
2. No celular, limpe cache/site data se já testou antes.
3. Recarregue a página 2 vezes.
4. Aguarde alguns segundos e use “Instalar app”.

Observação: se o navegador ainda mostrar apenas “Adicionar atalho”, normalmente é cache antigo, `sw.js` ausente na raiz, manifest não carregado, ou ícone 192/512 não encontrado.
